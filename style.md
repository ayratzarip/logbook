## Style guide (Moodtracker)

Этот документ описывает визуальный стиль проекта, чтобы переносить его в другие проекты и сохранять единообразие.

### Общие принципы

- **Mobile-first**: интерфейс должен одинаково хорошо работать в Telegram WebView на телефоне и на десктопе.
- **Минимализм**: максимум контраста/читабельности, минимум декоративных элементов.
- **Системные токены**: используйте только токены из Tailwind (`gray.*`, `brand.*`) и готовые CSS-компоненты (`.card`, `.btn-primary` и т.д.).
- **Тёмная тема**: включена через `darkMode: 'media'` — используйте `dark:` для критичных цветов.

### Шрифты

- **Основной шрифт**: `Inter` (подключается через Google Fonts), fallback: `system-ui, -apple-system, sans-serif`.
- **Сглаживание**: включено в `body` (`-webkit-font-smoothing`, `-moz-osx-font-smoothing`).

### Типографика (готовые классы)

Используйте классы из `src/index.css`:

- **Заголовок страницы**: `.text-h1` (1.125rem, 700)
- **Заголовки секций/карточек**: `.text-h2` (1rem, 700)
- **Основной текст**: `.text-body` (1rem, 400)
- **Подписи/пояснения**: `.text-caption` (0.875rem, 400, цвет `gray.60`)

Рекомендации:
- Внутри карточек заголовок → `text-h2`, описание → `text-caption`.
- Для важного текста используйте `text-body` + `font-semibold`.

### Цвета (Tailwind tokens)

Источник: `tailwind.config.js`.

#### Gray scale
- `gray.100` — белый
- `gray.95` — светлый фон приложения (light)
- `gray.90/85` — границы/вторичные поверхности
- `gray.60` — вторичный текст (caption)
- `gray.35/10/5` — тёмные поверхности/границы (dark)
- `gray.0` — чёрный

#### Brand (акцентный синий)
- `brand.70` — основной акцент (кнопки, активные состояния)
- `brand.10` — фон приложения в тёмной теме
- `brand.30/90` — вспомогательные оттенки (по необходимости)

### Фон и тема

- Light: фон приложения `bg-gray-95`, текст `text-gray-0`.
- Dark: фон приложения `dark:bg-brand-10`, текст `dark:text-gray-100`.

### Компоненты (CSS utility-классы проекта)

Определены в `src/index.css` в `@layer components`.

#### Карточки
- **`.card`**: `rounded-xl p-5 bg-white dark:bg-gray-5 shadow-sm`
- **`.card-lg`**: `rounded-2xl p-6 bg-white dark:bg-gray-5 shadow-sm`

Правило:
- Для основных блоков/экранов используйте `card-lg`, для вторичных — `card`.

#### Кнопки
- **`.btn-primary`**: акцентная кнопка (`bg-brand-70 text-white`, высота ≥ 48px, `active:scale-95`, `hover:opacity-90`)
- **`.btn-secondary`**: вторичная (`bg-gray-90 dark:bg-gray-35`, текст контрастный, та же высота)

Правило:
- Основное действие экрана — всегда `btn-primary`.
- Вторичные/отмены — `btn-secondary`.

#### Инпут
- **`.input`**: единый стиль текстовых полей + фокус (`focus:ring-2 focus:ring-brand-70`).

#### Списки
- **`.list-item`**: строка списка с нижней границей (последняя — без границы).

### Скругления, тени, отступы

- Скругления: `xl = 12px`, `2xl = 16px`.
- Тени: `shadow-sm`/`shadow-card` — лёгкие, без сильного blur.
- Минимальная высота интерактивных элементов: **48px** (`min-h-[48px]`).

### Layout

- Контейнер страницы: `Layout` задаёт фон (`bg-gray-95 dark:bg-brand-10`) и фиксирует области:
  - Header: **64px**
  - BottomNav (если включён): **64px**
- Контент внутри `main`: `max-w-screen-lg mx-auto px-4 py-6`.

### Навигация (BottomNav)

- Фон: `bg-white dark:bg-gray-5`
- Граница: `border-gray-90 dark:border-gray-35`
- Активный таб:
  - текст: `text-brand-70 font-semibold`
  - иконка: `opacity-100`
- Неактивный таб:
  - текст: `text-gray-60`
  - иконка: `opacity-40 grayscale`

### Графики (Recharts)

Общий стиль (ориентир):
- Сетка: светлая, `opacity ~ 0.2`, `strokeDasharray="3 3"`.
- Линия: акцентный синий (обычно `hsl(222, 76%, 60%)` / `brand`-похожий).
- Tooltip: светлая карточка в light и тёмная в dark, с лёгкой границей.

### Текст и тон

- В пользовательских текстах используйте **вежливую форму обращения на «Вы»**.
- Пишите коротко, без канцелярита; одна мысль — одно предложение.

### Что НЕ делать

- Не добавляйте новые «случайные» цвета вне палитры `gray/brand`.
- Не используйте разные стили кнопок/карточек в разных местах — берите `.btn-*` / `.card*`.
- Не делайте плотные/мелкие кликабельные элементы (держите 48px+).
